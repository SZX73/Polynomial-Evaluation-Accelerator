This test was to verify that writing to a memory address that already contains a polynomial correctly overwrites the old block with the new block.

This test begins by storing the polynomial 5x+5 into address 0 using the stp command. Then, it stores the polynomial 3x^2+2x+1 into address 0, overwriting the previous one. Finally the polynomial is computed with x=4, which outputs 57. Following the 57 is a 0, representing the STATUS_OK code.
