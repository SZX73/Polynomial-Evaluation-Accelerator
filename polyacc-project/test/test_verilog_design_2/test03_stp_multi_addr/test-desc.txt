The purpose of this test was to observe that the memory was able to correctlty handle multiple different polynomials at different address without corrupting the data.

It begins by storing the polynomial 3x^2+2x+1 at address 0. Then, it stores the polynomial 10x+5 at address 7. Then, it evaluates address 0's polynomial where x=2, resulting in 17. Following the 17 is a 0, representing a STATUS_OK message. Finally, it evaluates the address 7's polynomial at x=3, equaling 35, again with a 0 following the output.
